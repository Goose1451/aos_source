If you want your own instance, you can install it from here or configure AOS locally with Docker (Ubuntu OS is recommended) by running:
git clone https://github.hpe.com/avi-genshaft/new4Testing.git
cd new4Testing/Nimbus_aos/aos_all_in_one/
chmod +x startup.sh
./startup.sh

For the DevOps story:
git clone https://github.hpe.com/avi-genshaft/new4Testing.git
cd new4Testing/Nimbus_aos/aos_pipline/aos_all_in_one/
copy authorized_keys file to /root/.ssh on your machine (the host where you run the docker script from)
chmod +x startup.sh
./startup.sh

This release's source code can be downloaded from the same place in GitHub.

.NET client for admin actions and for tweaking the system is included in the installation. If you run it with docker, download the .NET client from here
